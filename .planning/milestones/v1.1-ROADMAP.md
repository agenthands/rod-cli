# Milestone v1.1: Stealth & Humanization

**Status:** ✅ SHIPPED 2026-06-21
**Phases:** 7-9
**Total Plans:** 3

## Overview

Adapt the `godoll` engine into `rod-cli` to provide robust bot detection evasion, realistic browser fingerprinting, and human-like interaction.

## Phases

### Phase 7: Stealth Browser Initialization

**Goal**: Hook up `godoll` stealth profile generation and launch the browser with it.
**Depends on**: Phase 6
**Plans**: 1 plan

Plans:

- [x] 07-01: Integrate godoll stealth browser initialization.

**Details:**
- Extract `godoll` stealth launcher and fingerprint generator packages.
- Ensure `rod-cli` spawns browsers that pass standard bot mitigation checks (WebGL/Navigator spoofing).

### Phase 8: Network Evasion

**Goal**: Inject dynamic headers matching the browser fingerprint.
**Depends on**: Phase 7
**Plans**: 1 plan

Plans:

- [x] 08-01: Inject stealth profile HTTP headers in the network router.

**Details:**
- Integrate `godoll` network interception to spoof `User-Agent`, `sec-ch-ua`, and `Accept-Language`.
- Ensure headers match the bayesian profile generated in Phase 7.

### Phase 9: Humanized Interactions

**Goal**: Replace instantaneous clicks and typing with human-like input flows.
**Depends on**: Phase 8
**Plans**: 1 plan

Plans:

- [x] 09-01: Replace native rod actions with humanize equivalents.

**Details:**
- Port `humanize` mouse and keyboard logic to replace `go-rod`'s native `.Click()` and `.Type()`.
- Implement bezier curve trajectories and realistic typing delays.

## Milestone Summary

**Key Decisions:**
- Decision: Use godoll module locally (Rationale: Native go integration prevents complex network bridging).
- Decision: Inject headers in rod-cli's router instead of godoll's (Rationale: Prevent HijackRouter conflicts).

**Issues Resolved:**
- Successfully evaluated `navigator.webdriver` to false using stealth generation.
- Resolved `mousemove` triggering accurately via Bezier path computation.

**Issues Deferred:**
- Deferred: wrapping `DblClick` with `humanize`. Since it is rarely used it does not block the milestone.

**Technical Debt Incurred:**
- Minimal. The humanize logic replaces raw actions effectively without heavy performance penalties.
