## Scope outcome

v1.7 set out as the project's most architecturally significant milestone (full-stack
evasion: CDP + TLS + profiles + hardening). During the run the **operator narrowed
scope**: TLS fingerprint spoofing was **cancelled** (real-Chrome-only constraint — rod-cli
drives real Chrome, whose TLS/JA3 is authentic by construction; TLS spoofing lives in a
separate project, "munch"). Phase 31 + TLS-01..04 moved to out-of-scope. This removed
the milestone's single largest risk and sharpened it to: **reduce CDP footprint, ship
curated Chrome-only profiles, and expand JS-layer hardening — all real Chrome.**
