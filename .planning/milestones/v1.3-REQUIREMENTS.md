# Requirements: v1.3 Godoll Migration

## Browser Management

- [ ] **BROWSER-01**: Implement `rod-cli install` command using `go-rod`'s built-in `launcher.NewBrowser()` to automatically fetch and cache Chromium, providing a zero-dependency setup (parity with Playwright).
- [ ] **BROWSER-02**: Replace the inline browser launcher setup in `context.go` with `godoll.NewBrowser(opts)` and configure it using `opts.StealthPreset()`.
- [ ] **BROWSER-03**: Replace the custom remote browser connection logic (`controlBrowser`) with `godoll/browser.ConnectToRemoteBrowser()`.

## Network Interception

- [ ] **NETWORK-01**: Replace raw `page.HijackRequests` routing logic in `rod-cli` with `godoll/network.NewInterceptor()` to handle request mocking and resource blocking (e.g. blocking all images via `**/*.jpg`).

## Humanized Interactions

- [ ] **HUMAN-01**: Migrate the raw `page.Mouse.Scroll(x, y, steps)` logic to `humanize.Scroll()` to implement physics-based bezier curve scrolling.

## Robust Execution

- [ ] **RETRY-01**: Wrap critical, flake-prone DOM actions (e.g. initial navigation, element resolution) with `godoll/retry`'s exponential backoff decorator to improve stability against transient network or rendering failures.
