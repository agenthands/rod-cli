<objective>
Make "vetted" real (PROF-02): a deterministic gate that runs EVERY built-in profile
through the v1.6 consistency validator and the offline detection harness, so an
incoherent or detectable profile fails the build and is not shipped. Document the
library in docs/stealth-config.md.

Purpose: the trust anchor of the whole phase. Without this gate, "vetted profiles"
is just a claim; with it, a future edit that breaks a profile's coherence turns the
build red. Deterministic + offline so it lives in the default CI gate.
</objective>
